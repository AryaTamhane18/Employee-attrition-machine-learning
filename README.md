# Employee Attrition Prediction using Machine Learning

## Project Overview
This project analyzes employee data to predict employee attrition and understand salary patterns using machine learning models.

The project includes both classification and regression tasks and compares multiple models to evaluate their performance.

## Dataset
IBM HR Analytics Employee Attrition Dataset

- 1470 employee records
- 35 features related to employee demographics, job role, and work environment

## Objectives
- Predict whether an employee will leave the company (classification)
- Predict monthly income using employee attributes (regression)
- Compare multiple machine learning models
- Analyze model performance and interpret results

## Machine Learning Models Used

### Classification
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Decision Tree
- Random Forest
- Neural Network (MLP)

### Regression
- Linear Regression

### Unsupervised Learning
- Principal Component Analysis (PCA)

## Evaluation Metrics
Models were evaluated using:

- Accuracy
- Precision
- Recall
- F1-score
- Mean Squared Error (MSE)
- R² Score
- Cross-validation

## Results
- Attrition prediction accuracy: ~89%
- Salary prediction performance: R² ≈ 0.89
